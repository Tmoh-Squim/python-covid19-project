# COVID-19 Global Data Tracker

## Project Description
This project is a Python-based data tracker that collects, analyzes, and visualizes COVID-19 global statistics.  
It provides up-to-date insights on cases, recoveries, and deaths worldwide using public APIs and data sources.

## Objectives
- Fetch and process real-time COVID-19 data from trusted APIs  
- Analyze trends across countries and continents  
- Visualize data using charts and graphs for easy interpretation  
- Create a user-friendly interface to explore key statistics

## Tools and Libraries Used
- Python 3.x  
- pandas (data manipulation)  
- matplotlib / seaborn (data visualization)  
- requests (API data fetching)  
- Jupyter Notebook (development environment)
